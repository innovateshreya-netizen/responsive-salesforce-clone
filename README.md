# Responsive Salesforce Clone

A responsive Salesforce homepage clone built using HTML and CSS to practice frontend web development fundamentals and gain hands-on experience with real-world website layouts.

## Features

- Salesforce-inspired navigation bar
- Responsive landing page layout
- Hero section with call-to-action buttons
- Product showcase cards
- Fixed header design
- Clean and modern user interface
- Flexbox-based layout structure
- Integrated Remix Icons

## Technologies Used

- HTML5
- CSS3
- Remix Icons

## Purpose

This project was developed as my first frontend project to apply and strengthen my understanding of:

- HTML page structure and semantic elements
- CSS styling and layout techniques
- Flexbox for responsive design
- Positioning and alignment
- Responsive web design principles
- UI recreation from real-world websites
- Project organization and code management

## Project Preview

A Salesforce-inspired landing page featuring a responsive navigation menu, promotional hero section, product cards, and modern UI components designed to closely resemble the original website.
